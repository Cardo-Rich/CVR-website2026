Labeled form control with the brand's rose focus ring. One component for text inputs, textareas, and selects.

```jsx
<Field label="Full name" placeholder="Jane Owner" />
<Field label="Neighborhood" as="select" options={["La Jolla","Pacific Beach","Del Mar"]} />
<Field label="Tell us about your home" as="textarea" />
```

Switch type with `as="textarea" | "select"`; pass `options` for selects.
